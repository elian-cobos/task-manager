# Task Manager (Java)

Aplicación de consola desarrollada en Java para gestionar tareas.

## Funcionalidades
- Crear tareas
- Listar tareas
- Marcar como completadas
- Eliminar tareas

## Tecnologías
- Java
- Programación Orientada a Objetos
- Colecciones (ArrayList)

## Cómo ejecutar
Compilar y ejecutar el archivo Main.java
